### GUIA DE INSTALAÇÃO DO BOT (TERMUX)

**Primeiro, instale o Termux:**
É muito importante que você instale o Termux pela loja F-Droid. A versão da Play Store está desatualizada e não vai funcionar.
*   Acesse este link e baixe o app: https://f-droid.org/pt_BR/packages/com.termux/

**Agora, execute os comandos de instalação:**
Abra o Termux e siga a numeração, um comando de cada vez:

1.  `pkg update && pkg upgrade -y`
2.  `pkg install python git screen -y`
3.  `pip install --upgrade pip`
4.  `git clone https://github.com/tumioka8-design/botvenda.git`
5.  `cd botvenda`
6.  `pip install -r requirements.txt`
7.  `screen -S bot`
8.  `python bot.py`

**Para deixar o bot rodando em segundo plano:**
Após o último comando, o bot estará online. Para mantê-lo funcionando, pressione a seguinte combinação de teclas:
*   Ctrl + A (pressione e solte)
*   Depois, a tecla D

Pronto! Agora você pode fechar o Termux.

---
### COMANDOS ÚTEIS

*   **Para ver o que o bot está fazendo:**
    `screen -r bot`

*   **Para parar o bot (dentro da tela do bot):**
    Pressione Ctrl + C

*   **Para atualizar o bot:**
    `cd botvenda`
    `git pull`
